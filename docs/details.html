Initial Setup

System Updates

Update Package Lists and System 
# Update package lists and upgrade system
sudo apt update && sudo apt upgrade -y

Always start by updating your system to ensure you have the latest packages and security updates.

Network Tools Installation

Install Network Utilities 
# Install essential network tools
sudo apt install -y net-tools

# Verify installation and check network configuration
ifconfig
netstat -tuln

Network tools are essential for diagnosing connectivity issues and monitoring network interfaces.

ROS Master Configuration in Docker

Linux Host Network Mode: For seamless ROS communication between containers on Linux systems.

‚ö° Master Container Setup

Start ROS Master 
# Start ROS Master container
docker run -it --network host --name ros-master ghcr.io/remasteredgod/ros-noetic:20.04

# Inside the container, start roscore
roscore

The master container runs roscore and coordinates all ROS communication.

üîó Slave Container Setup

Start ROS Slave 
# Start ROS Slave container
docker run -it --network host --name ros-slave ghcr.io/remasteredgod/ros-noetic:20.04

# Inside the container, configure ROS environment
export ROS_MASTER_URI=http://127.0.0.1:11311
export ROS_IP=127.0.0.1

The slave container connects to the master and can run ROS nodes.

üìù Persistent Configuration with .bashrc

Configure .bashrc in Slave Container 
# Location of bash startup file
# Full path: /root/.bashrc (in Docker containers)
~/.bashrc

# Open the file for editing
nano ~/.bashrc

# If nano is not installed, install it first:
apt-get update && apt-get install -y nano

# Alternative editors:
vi ~/.bashrc
Add ROS Settings to .bashrc 
# Add these lines at the bottom of ~/.bashrc
export ROS_MASTER_URI=http://127.0.0.1:11311
export ROS_IP=127.0.0.1

# Reload the configuration
source ~/.bashrc

# Verify the settings
echo $ROS_MASTER_URI
echo $ROS_IP

üí° Pro Tip: Adding ROS environment variables to ~/.bashrc ensures they're automatically loaded every time you start a new terminal session in the container.
