Quick Start

Ubuntu 20.04 Base Image

Pull and Run Ubuntu Base 
# Pull the image
docker pull ghcr.io/remasteredgod/ubuntu-focal:20.04

# Run interactive container with host networking
docker run -it --network host ghcr.io/remasteredgod/ubuntu-focal:20.04

Building from Source

Prerequisites: Docker Desktop with buildx enabled and GitHub Container Registry access

Build Ubuntu Base

Build Base Image 
docker build -t ghcr.io/remasteredgod/ubuntu-focal:20.04 .

Build ROS Noetic (Multi-Platform)

Multi-Platform Build 
docker buildx build \
  --platform linux/amd64,linux/arm64 \
  --build-arg BASE_IMAGE=ghcr.io/remasteredgod/ubuntu-focal:20.04 \
  -t ghcr.io/remasteredgod/ros-noetic:20.04 \
  -f Dockerfile.ros \
  --push .

Configuration

Environment Variables

Variable Default Description
DEBIAN_FRONTEND noninteractive Prevents interactive prompts during package installation
TZ UTC System timezone configuration
LANG en_US.UTF-8 System locale and character encoding
ROS_DISTRO noetic ROS distribution version (ROS image only)

Build Arguments

Custom Base Image 
# Use custom base image
docker build --build-arg BASE_IMAGE=custom-ubuntu:20.04 -f Dockerfile.ros .

Troubleshooting

Build Issues

Interactive Prompts During Build

Problem: Build hangs waiting for timezone/locale selection

Solution: Ensure DEBIAN_FRONTEND=noninteractive is set

ENV DEBIAN_FRONTEND=noninteractive

Package Repository Errors

Problem: 400 Bad Request from archive.ubuntu.com

Solution: Uses kernel.org mirrors automatically

Runtime Issues

ROS Commands Not Found

Problem: roscore command not available

Solution: Use ROS image, not base Ubuntu image

docker pull ghcr.io/remasteredgod/ros-noetic:20.04

File Permission Issues

Problem: Cannot write to mounted volumes

Solution: Match user IDs

docker run --user $(id -u):$(id -g) ...
